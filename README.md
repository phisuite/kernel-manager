## Overview

This project contains the tools to orchestrate the different services.

## Commands

Start services:
```bash
make start
```

Update services:
```bash
make update
```

Stop services:
```bash
make stop
```

List all running services
```bash
make list
```

Show the logs of the service TARGET
```bash
make log TARGET=<service-name>
```
